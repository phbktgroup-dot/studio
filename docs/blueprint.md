# **App Name**: PHBKT Group Limited

## Core Features:

- Hero Section Animation: Implement a blur-to-clear scroll animation for the hero section with a rotating 3D blue sphere.
- Service Showcase: Display services with a vertical floating card featuring a 3D mobile app mockup with a magnetic hover effect.
- Marketing Section: Showcase a marketing section with a glowing abstract network graph and text reveal on scroll.
- Compliance Section: Design a tax services section with a minimalist 3D legal seal icon and a clean layout.
- Business Launchpad: Create a business registration section with 3D geometric building blocks.
- Interactive Roadmap: Implement a vertical scroll-linked roadmap section with a moving frame and glowing milestones. that uses tool.
- Real-time Stats: Display a live, ticking counter for total clients, pulling data from the Supabase database.
- Login Authentication: Create a login authentication view with a minimalist logo using Supabase Authentication.
- Client Dashboard: Build a dashboard with side-navigation and data from the Supabase database.

## Style Guidelines:

- Primary color: Royal Blue (#4169E1) to represent trust and innovation.
- Background color: Ghost White (#F8F8FF), creating a clean, professional 'White-Pro' aesthetic.
- Accent color: Slate Gray (#708090) for a modern, understated feel. It should be of different brightness than Royal Blue.
- Headline font: 'Mukta' (sans-serif) for Marathi quotes.
- Body font: 'Inter' (sans-serif) for English sub-text.
- Use minimalist 3D icons inside white frames for each section.
- Employ a clean layout with ample whitespace and subtle magnetic hover effects on central frames.
- Incorporate scroll animations, staggered letter-reveal effects, and a bobbing 3D mailbox icon.
